# RH Onboarding Automation

Automação do processo de onboarding de colaboradores.

## Objetivo
- Centralizar o controlo do onboarding
- Automatizar criação de dossier, documentos e tarefas
- Garantir auditabilidade e controlo de estado

## Estrutura
- /flows → Exportações JSON dos flows Power Automate
- /docs → Documentação do processo e estados

## Fonte da Verdade
- O Git é a fonte da verdade dos flows
- Power Automate é apenas o runtime

## Estados do Processo
Definidos na lista SharePoint `Checklist_Onboarding`.
